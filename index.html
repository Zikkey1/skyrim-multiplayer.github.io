Skyrim Multiplayer

Skyrim Multiplayer (SkyMP) is a modification for Skyrim SE allowing to play online.

< DOWNLOAD BUTTON >

Goals

  • Stable, full playable coop-mode with working main quest.
  • DLC and global modifications support (like Beyond Skyrim).
  • Support as many players in a single world as possible (up to 1000).
  • Plugin system that allows writing clientside and serverside extensions in JavaScript/TypeScript. It would allow server owners to implement unique game mechanics (guilds, factions, jobs, combat or social interaction between players, etc). Possibilities would be limited only by your programming skills.
  • Emulate Bethesda's Papyrus-based scripting system on the server-side to synchronize things written in Papyrus like ore mining, nirnroots, some animals, etc.
  • Embedded anticheat, for opening public servers without fear of cheaters.
  • Chromium Embedded Framework for displaying implemented from scratch interfaces like chat, party, admin panel, guild panel, minigames, or whatever you can imagine. This will make it possible to use modern Web technologies in Skyrim.

Contributing

At the moment, almost all parts of our source code are closed-source. We are comfortable working privately as a small team. But obviously, we will be publishing API to be used freely to keep our community decentralized and fill Skyrim of different game modes. If you believe we should face to open-source, you are interested in helping to code or you have any questions, please contact us.
